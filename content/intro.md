# Welcome to Slides

A beautiful, markdown-powered presentation tool.

Press **→** or **Space** to advance.

---

## Write in Markdown

- Author your slides in plain `.md` files
- Separate slides with a horizontal rule (`---`)
- Use headings, lists, code blocks, and images

---

## Multiple Decks

Each file in the `content/` folder becomes its own deck.

Add a new `.md` file and it appears on the homepage automatically.

---

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `→` / `Space` | Next slide |
| `←` | Previous slide |
| `F` | Toggle fullscreen |
| `Esc` | Exit fullscreen |

---

## Light & Dark Mode

Click the 🌙 / ☀️ button in the top-right corner to switch themes.

Your preference is saved automatically.

---

## Get Started

1. Edit files in `content/`
2. Run `npm run dev`
3. Present!

That's it. Happy presenting!
