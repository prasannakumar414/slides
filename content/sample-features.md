# Feature Showcase

Exploring everything you can put on a slide.

---

## Rich Text

You can use **bold**, *italic*, ~~strikethrough~~, and `inline code`.

> Blockquotes work great for callouts and quotations.

---

## Code Blocks

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```

Syntax highlighting keeps code readable.

---

## Lists

### Unordered

- React + Vite for fast builds
- Markdown for easy authoring
- CSS custom properties for theming

### Ordered

1. Write your content
2. Drop it in `content/`
3. Present it anywhere

---

## Links & Images

Visit the [Vite docs](https://vitejs.dev) or the [React docs](https://react.dev) for more.

Images are supported too — just use standard markdown syntax:

```
![Alt text](https://example.com/image.png)
```

---

## That's a Wrap

Add more slides by creating new `.md` files.

Each file is a self-contained presentation deck.
